# Caterpillar Hundred Square

A simple tablet-first adding and subtracting game on a hundred square. The child taps the destination square and the caterpillar moves there when the answer is correct.

## Local development

From the project root, serve the static files with:

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000`.

## Deployment

The site is designed to deploy directly as a static Netlify site with the publish directory set to the repository root.
